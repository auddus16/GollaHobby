## 💡 Features

### 1. Video tutorial
  * `VideoView`를 활용한 동영상 튜토리얼 제공

### 2. ViewPager
  * `ViewPager`를 활용한 탭 레이아웃(`tab layout`) 구성


### 3. SeekBar
  * `SeekBar`를 정도 측정 및 진행 현황 표시


### 4. RadarChart
  * `RadarChart`를 이용하여 수행 결과를 그래프 형태로 표시
